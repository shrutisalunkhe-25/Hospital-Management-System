# Hostel-Management-System
First Database project

## 1. Project Discription:
The Hostel Management System is a database project devoloped using mysql.
<br>
It manage and stores hostel details,students records,room information,payment records and visitor records.
<br>
----

## 2. Features:
- Student Management
- Room allocation
- Fee tracking 
- Visitor Records
<br>
----

## 3. Objective:
- To reduce manual hostel records management
- TO implement One to Many relationship
- To apply Constraints like check 
- To understand real world application of database concepts
<br>
----

## 4. Entities used in project:
1. Hostel
2. Room
3. Student
4. Payment
5. Visitor
<br>
----

## 5. DBMS concepts used in project:
- Primary key
- Foreign key
- One to Many relationship
- CHECK costraint
- Aggregate Functions(SUM, COUNT)
<br>
----
